## Download Instructions

### Manual Download
1. Click on any photo URL above
2. Click the green "Free Download" button
3. Select desired resolution (recommended: Original size or Large 1920px)
4. Save to `/public/images/stock/[category]/`

### Using Pexels API (Recommended for Bulk)
```bash
# Get API key from: https://www.pexels.com/api/
# Documentation: https://www.pexels.com/api/documentation/

# Example cURL request
curl -H "Authorization: YOUR_API_KEY" \
  "https://api.pexels.com/v1/photos/3184360"
```

### Image Optimization
After downloading, optimize images for web:

```bash
# Using sharp-cli (Node.js)
npx sharp-cli resize 1920 --withoutEnlargement --format webp input.jpg output.webp

# Using ImageMagick
convert input.jpg -resize 1920x -quality 85 output.webp
```

Recommended formats:
- **WebP** - Primary format (best compression/quality ratio)
- **AVIF** - For modern browsers (even better compression)
- **JPG** - Fallback for older browsers
